Visual Search Application

This guide will help you set up and run the Visual Search Application on your local machine.

Step 1: Clone the Repository

git clone https://github.com/abdoula666/virsualsearch.git
cd virsualsearch

Step 2: Set Up Python Environment

Create and activate a virtual environment:

For Windows:

python -m venv .venv
.venv\Scripts\activate

For Linux/Mac:

python -m venv .venv
source .venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Step 3: Configure WooCommerce Credentials

  1. Copy .env.example to .env
  2. Open .env and add your WooCommerce credentials: 
    WOOCOMMERCE_URL=your_store_url
WOOCOMMERCE_CONSUMER_KEY=your_key
WOOCOMMERCE_CONSUMER_SECRET=your_secret
⚠️ Never share or commit your .env file containing real credentials!

Step 4: Add Product Images

  1. Place your product images in the product_images folder
  2. Use the naming format: productID_description.jpg
  3. Ensure productID matches your WooCommerce product IDs

Step 5: Generate Feature Vectors

python feature_extraction.py

This step may take a few minutes depending on the number of images.

Step 6: Run the Application

python app.py

Open your browser and navigate to: http://localhost:5000

Troubleshooting

  • Make sure all dependencies are installed correctly
  • Verify your WooCommerce credentials are correct
  • Check that your product images are in the correct format
  • Ensure your WooCommerce API access is enabled

For more detailed information and updates, visit the GitHub repository.
